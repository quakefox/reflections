When would you want to use a remote repository rather than keeping all your work
local?

    If I am working on shared work with other people or want to use several computers to work on the same project.

Why might you want to always pull changes manually rather than having Git
automatically stay up-to-date with your remote repository?

    If another person commit sth which might result in a conflict. Manual pull controls the local environment.

Describe the differences between forks, clones, and branches.  When would you
use one instead of another?

    Clone is to get everything on a different device. Branch is to create a variant on the repo to do parallel work and merge back. Fork is like clone when you don't have access to the repo.
